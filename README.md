## js-xlsx demo
Javascript 解析、导入、导出 Excel 文件小示例。包括了 web（仅支持现代浏览器） 和 nodejs（需要 v7.6+）。

```bash
git clone https://github.com/Lizhooh/js-xlsx-demo.git
```

![](./_/view.png)

## use

**web:**

```bash
cd web
npm install -g serve
serve ./ # open browser http://127.0.0.1:5000
```

**nodejs:**

```bash
cd node
npm install
npm start
```
